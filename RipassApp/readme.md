# RipassApp

RipassApp è una pagina web progettata per facilitare il ripasso di argomenti disciplinari, con un'interfaccia semplice e ottimizzata per dispositivi mobili.

## Come funziona
- Ogni domanda è contenuta in una "card".
- Cliccando sul pulsante "Mostra risposta", la risposta viene visualizzata.
- Cliccando di nuovo, la risposta viene nascosta.

## Struttura del progetto
- `index.html` → struttura della pagina
- `style.css` → stili e layout
- `script.js` → gestione dell'interazione utente
- `LICENSE` → licenza Creative Commons

## Installazione e uso
1. Scaricare o clonare il repository.
2. Aprire `index.html` in un browser.
3. Aggiungere altre domande duplicando le card in HTML.

## Tecnologie utilizzate

- CSS
